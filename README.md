#  Advanced Curved Lane Detection System

A **Research-Grade Computer Vision System** for Robust Lane Detection in Autonomous Driving Applications.

---

##  Overview
This project implements an **advanced curved lane detection system** using **Python** and **OpenCV**.  
It provides **real-time**, **accurate**, and **robust lane detection** under varying lighting conditions and curved road geometries.

---

###  Requirements
- **Python 3.8+**
- **OpenCV 4.5+**
